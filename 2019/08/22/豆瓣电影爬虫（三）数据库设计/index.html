豆瓣电影爬虫（三）数据库设计

2019年08月22日 60 字 大概 1 分钟 爬虫 数据库 爬虫 豆瓣

将爬虫分析到的10个字断存储在数据库中。

create database douban;

create table movie(
    id int primary key not null,
    title varchar(200) not null,
    directors varchar(100),
    star int,
    rate int,
    casts varchar(200),
    cover varchar(100),
    cover_x varchar(100),
    cover_y varchar(100),
    url varchar(200))default charset =utf8
